# Customer Segmentation Analysis

## Project Overview
This project uses machine learning techniques to group customers into different segments based on their purchasing behavior and demographic information.

## Features
• Customer Data Analysis
• Data Preprocessing
• K-Means Clustering
• Customer Group Identification
• Data Visualization
• Marketing Insights Generation

## Technologies Used
• Python
• Pandas
• NumPy
• Matplotlib
• Scikit-learn
• Google Colab
